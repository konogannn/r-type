# POC Documentation

Ce dossier contient les preuves de concept (POC) pour le projet R-Type. Chaque POC est conçu pour démontrer une fonctionnalité clé du projet.

## Liste des POC

1. **Serveur multithreadé**
   - Objectif : Démontrer la gestion de plusieurs clients via un serveur UDP.
   - Fichier associé : `server_poc.md`

2. **Client graphique**
   - Objectif : Afficher un fond étoilé défilant et permettre le contrôle d'un vaisseau.
   - Fichier associé : `client_poc.md`

3. **Protocole réseau**
   - Objectif : Définir et tester un protocole binaire pour la communication client-serveur.
   - Fichier associé : `protocol_poc.md`

4. **Accessibilité**
   - Objectif : Ajouter des fonctionnalités pour améliorer l'accessibilité du jeu.
   - Fichier associé : `accessibility_poc.md`

---

Chaque fichier détaille les objectifs, les étapes de mise en œuvre, et les résultats attendus pour le POC correspondant.

## Sécurité (C7)

5. **Security Review**
   - Objective: Regularly review protocols/components against new vulnerabilities; define mitigations and standards.
   - File: `security_review.md`

6. **Security Watch Process**
   - Objective: Ongoing monitoring workflow (sources, cadence, triage) and evidence of actions.
   - File: `security_watch.md`
